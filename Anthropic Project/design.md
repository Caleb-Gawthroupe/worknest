# Worksheet Creator - Design System

## Design Philosophy

### Color Palette
- **Primary**: Deep slate blue (#1e293b) - Professional, trustworthy, academic
- **Secondary**: Warm amber (#f59e0b) - Creative, energetic, highlights
- **Accent**: Soft sage green (#10b981) - Success, growth, educational
- **Neutral**: Cool gray (#64748b) - Supporting text, borders
- **Background**: Pure white (#ffffff) - Clean, academic paper feel

### Typography
- **Display Font**: "Tiempos Headline" - Bold serif for headings, evokes academic authority
- **Body Font**: "Inter" - Clean sans-serif for UI elements and body text
- **Code Font**: "JetBrains Mono" - For LaTeX code display

### Visual Language
- **Academic Excellence**: Clean, professional interface that inspires confidence in educational quality
- **Modern Simplicity**: Minimalist design that reduces cognitive load for educators
- **Collaborative Feel**: Subtle indicators of real-time collaboration and feedback
- **Precision**: Sharp, exact visual elements that mirror the precision of LaTeX typesetting

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and state transitions
- **ECharts.js**: Progress visualization and analytics charts
- **p5.js**: Generative background patterns and creative coding elements
- **Splitting.js**: Text animation effects for headings
- **Splide.js**: Template carousel and image galleries

### Animation & Effects
- **Text Effects**: 
  - Split-by-letter stagger animation for main headings
  - Subtle color cycling on "AI-powered" text
  - Typewriter effect for LaTeX code preview
- **Background**: 
  - Subtle geometric pattern generation using p5.js
  - Floating particle system suggesting knowledge networks
- **Interactive Elements**:
  - 3D tilt effect on cards and buttons
  - Smooth morphing transitions between states
  - Real-time progress indicators with fluid animations

### Header Effect
- **Generative Background**: Animated network of connected nodes representing knowledge concepts
- **Gradient Overlay**: Subtle linear gradient from primary to secondary color
- **Parallax Scrolling**: Gentle depth effect on background elements

### Component Styling
- **Cards**: Soft shadows with subtle hover lift effects
- **Buttons**: Gradient backgrounds with smooth hover transitions
- **Form Elements**: Clean borders with focus state animations
- **Chat Interface**: Message bubbles with typing indicators
- **PDF Preview**: Clean iframe styling with loading states

### Responsive Design
- **Desktop**: Full dual-panel layout with side-by-side preview
- **Tablet**: Collapsible panels with swipe navigation
- **Mobile**: Stacked layout with bottom sheet chat interface

### Accessibility
- **High Contrast**: 4.5:1 minimum contrast ratio for all text
- **Focus Indicators**: Clear keyboard navigation states
- **Screen Reader**: Proper ARIA labels and semantic HTML
- **Motion Preferences**: Respect user's reduced motion settings
